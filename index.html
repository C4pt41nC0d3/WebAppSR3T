El monitoreo de los robots SR3T es una de las características más releventates, en esta aplicación web.
El muestreo de los datos de los robots SR3T, permite al usuario hacer uso de las variables sensoriales y de posición del robot.
La geolocalización de los robots, es una caractertística importante de la aplicación, esto se logra mediante el API de google maps.
La arquitectura de esta aplicación, está programada en PHP con el microframework Silex.
